# First Glance at the Menu

Written by Admin
Now if you aren't familiar with Null Core or you've never seen it, let's start with its actual User Interface (later called UI). On the image below you can see the first appeareance of it once you have it injected in game by clicking INS.
Null Core will always remind you the button to open the cheat once you load in TF2 with it by notification
At the bottom you'll see navigation tab and little icon on the left you have to click to open menu
Upon clicking on the icon, you'll see the menu with tabs and quick-acess console commands for your own comfort pop up. Let's break it down - no need to hurry.
Let's call it navigation menu
Here you can change your steam name, custom name and disconnect reason, quickly reconnect to the server you're currently on (Retry button), fix your bugged in-game sound, i.e non-stopping pyros flamethrower sounds, sounds of cart moving even if it doesn't (SND_Restart button or StopSound which would be a faster option). CL_FullUpdate button can also fix various things like elements of your HUD (health, ammo) being invisible for some unknown reason and Status button which will show you information about players on the server, their ping, steam IDs in console and so on. You can also notice information about Cloud Usage being shown. You see - Null Core also stores your configuration files and visual configurations in cloud, which then gets synchronized with the cloud upon closing TF2 or opening it.
Please note that Custom Name and Disconnect Reason no longer work in official TF2 servers
